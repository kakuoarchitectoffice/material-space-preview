# Material Space Private Preview

タイルと空間デザインを紹介する、限定公開用の静的コンセプトサイトです。

## ローカル確認

依存パッケージはありません。`index.html` をブラウザで開くか、任意の静的HTTPサーバーで配信してください。

```bash
python3 -m http.server 4173
```

## テスト

```bash
npm test
```

## アクセス制限について

GitHub Pagesは静的ホスティングのため、このパスワード画面は閲覧者を限定するための簡易ゲートです。サーバー側認証ではないため、機密情報は配置しないでください。検索エンジン向けには `noindex` を設定しています。
